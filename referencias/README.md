# Referências Bibliográficas

Esta pasta contém as referências teóricas utilizadas no desenvolvimento do projeto de análise da hierarquia de memória.

## Principais referências

- Hennessy, J. L.; Patterson, D. A.  
  *Computer Architecture: A Quantitative Approach*.  
  (Hierarquia de memória, caches e desempenho)

- McCalpin, J. D. (1995)  
  STREAM Benchmark – Medição de largura de banda de memória

- Intel Corporation  
  Documentação técnica do processador Intel Core i5-1235U  
  (Arquitetura híbrida e níveis de cache)

## Conceitos abordados

- Hierarquia de memória (L1, L2, L3, RAM)
- Localidade temporal e espacial
- Largura de banda de memória
- Cache miss e cache hit
